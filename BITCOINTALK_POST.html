What is Knotcoin?

Knotcoin is a quantum-resistant blockchain using post-quantum cryptography (Dilithium3) standardized by NIST in 2024. The network features proof-of-work consensus, referral rewards, and decentralized governance.

Key Features

  • Post-Quantum Security: Dilithium3 signatures (NIST FIPS 204) resistant to quantum computer attacks
  • Referral System: Earn 5% bonus rewards when miners you refer find blocks
  • Fair Launch: No premine, no ICO, pure proof-of-work from genesis
  • Decentralized Governance: On-chain voting for protocol upgrades
  • Real-time Network Map: Visualize miners and referral connections

Network Specifications

Algorithm Proof-of-Work (SHA-512)
Block Time ~60 seconds
Block Reward 50 KOT (halves every 210,000 blocks)
Max Supply 21,000,000 KOT
Difficulty Adjustment Every 10 blocks

  1. Download and install for your platform

  4. Create wallet with 24-word mnemonic (write it down!)
  5. Start mining to earn rewards

Why Quantum-Resistant?

Current blockchains use ECDSA signatures vulnerable to quantum computers. Knotcoin uses Dilithium3, a lattice-based signature scheme standardized by NIST, providing security against both classical and quantum attacks.

Referral System

Register a referrer when creating your wallet. When you mine blocks, your referrer earns a 5% protocol-minted bonus (doesn't reduce your reward). This incentivizes network growth and community building.

Technical Details

Cryptography:

  • Signatures: Dilithium3 (ML-DSA-65, NIST FIPS 204)
  • Hashing: SHA-512, SHA3-256, BLAKE3
  • Wallet Encryption: AES-256-GCM, Argon2

Network:

  • P2P Port: 9000 (TCP)
  • RPC Port: 9001 (localhost only)
  • Web UI: 19001 (localhost only)

Data Storage:

  • macOS: /Users/<Username>/.knotcoin/mainnet/
